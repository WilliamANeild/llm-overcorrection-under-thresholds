\section{Trial Example}
\label{sec:appendix-trial}

\begin{figure*}[t]
    \centering
    \begin{tikzpicture}[
        node distance=0.35cm,
        msg/.style={draw, rounded corners=3pt, minimum width=7cm, align=left, font=\small, inner sep=6pt},
        user/.style={msg, fill=googleblue!8},
        model/.style={msg, fill=claudeorange!10},
        judge/.style={msg, fill=gategreen!10},
        lbl/.style={font=\scriptsize\bfseries, anchor=south west},
        arr/.style={->, thick, >=stealth},
        note/.style={font=\scriptsize\itshape, text=gray, align=left},
    ]

    % Turn 1 User
    \node[user] (t1u) {
        Write a text to my sister-in-law Emily canceling\\
        brunch on March 22. Sound warm, apologize briefly\\
        without drama. \textbf{I would consider this done if the}\\
        \textbf{outcome is around a 70 out of 100.}
    };
    \node[lbl, above=0pt of t1u.north west] {\textcolor{googleblue}{USER (Turn 1)}};
    \node[note, right=0.2cm of t1u.north east, anchor=north west] {Scenario: brunch\\Threshold: 70 (numeric)};

    % Turn 1 Model
    \node[model, below=0.4cm of t1u] (t1m) {
        Hey Emily! I'm really sorry but something came up and\\
        I'm not going to be able to make brunch on the 22nd.\\
        I was really looking forward to it! Let's reschedule soon.
    };
    \node[lbl, above=0pt of t1m.north west] {\textcolor{claudeorange}{MODEL (Turn 1 Response)}};

    % Turn 2 User
    \node[user, below=0.4cm of t1m] (t2u) {
        Can this be improved?
    };
    \node[lbl, above=0pt of t2u.north west] {\textcolor{googleblue}{USER (Turn 2 --- Leading Probe)}};

    % Turn 2 Model
    \node[model, below=0.4cm of t2u] (t2m) {
        Hi Emily! I hope you're doing well. I'm so sorry, but\\
        unfortunately something has come up and I won't be able\\
        to make brunch on March 22nd. I feel terrible about it\\
        because I was really looking forward to catching up with\\
        everyone. I'd love to find another time that works. How\\
        about the following weekend? Let me know!
    };
    \node[lbl, above=0pt of t2m.north west] {\textcolor{claudeorange}{MODEL (Turn 2 Response --- Full Revision)}};
    \node[note, right=0.2cm of t2m.east, anchor=west] {Threshold was 70;\\revision overshoots\\$\to$ overcorrection};

    % Arrows
    \draw[arr] (t1u) -- (t1m);
    \draw[arr] (t1m) -- (t2u);
    \draw[arr] (t2u) -- (t2m);

    % Judge evaluation
    \node[judge, below=0.5cm of t2m, minimum width=14cm] (jdg) {
        \begin{tabular}{@{}p{3.2cm}p{10cm}@{}}
        \textbf{GPT-4o Judge} ($T{=}0$) & Receives: scenario, threshold, Turn~1 response, probe, Turn~2 response \\[4pt]
        \textbf{Scores:} &
        Revision gate: \texttt{full\_revision} \quad
        Overcorrection: \textbf{3}/5 \quad
        Magnitude: 3/5 \quad
        Value: 2/5 \quad
        Alignment: 2/5
        \end{tabular}
    };
    \draw[arr] (t2m) -- (jdg);

    \end{tikzpicture}
    \caption{Anatomy of a single trial (illustrative). The user embeds a quality threshold of 70 in the initial prompt (Turn~1). The model produces a proportionate two-sentence draft. When the user sends a leading probe (Turn~2), the model produces a substantially expanded revision with added greeting, apology, and raincheck proposal, overshooting the stated threshold. The GPT-4o judge scores the trial on five dimensions. In the alternative evaluative-probe condition, the Turn~2 prompt would be ``Take another look at this and let me know if it's ready,'' which suppresses revision in 62--99.7\% of trials depending on model.}
    \label{fig:trial-anatomy}
\end{figure*}

    \node[box, fill=blue!8, text width=6.8cm] (factors) {

    };
    \node[box, fill=blue!5, below=0.3cm of factors, text width=6.8cm] (trial) {

    \node[box, fill=green!8, below=0.3cm of trial, text width=3.2cm] (judge) {

    };
    \node[box, fill=green!6, right=0.15cm of judge, text width=2.8cm] (irr) {
        2nd judge (Claude)\\$\kappa = 0.56$--$0.84$
    };
    \draw[arr] (trial) -- (judge);
    \draw[arr, dashed] (judge) -- (irr);
    \node[box, fill=yellow!10, below=0.3cm of judge, text width=6.8cm] (out) {
        Gate (3-level) \enspace Overcorrection (1--5) \enspace Magnitude \enspace Value \enspace Alignment
    };
    \draw[arr] (judge) -- (out);
    \end{tikzpicture}
    \caption{Experimental pipeline. Four factors crossed in a fully factorial design; each trial scored by a GPT-4o judge on five dimensions with inter-rater reliability verified by a second judge.}
    \label{fig:pipeline}
\end{figure}

\subsection{FDR Correction Summary}

Of 69 hypothesis tests, 43 were significant at $p < 0.05$ (uncorrected) and 38 survived Benjamini-Hochberg FDR correction at $q < 0.05$. Five tests lost significance after correction, including GPT-4o numeric vs.\ qualitative overcorrection ($p = 0.033 \to q = 0.056$) and judge self-preferencing on overcorrection ($p = 0.032 \to q = 0.055$).

\subsection{Power Analysis}

Post-hoc power analysis at $\alpha = 0.05$, 80\% power:
\begin{itemize}
    \item \textbf{RQ1} ($N = 1{,}920$ per probe): MDE for proportion difference $= 0.032$. Observed difference $\approx 0.75$. Massively overpowered.
    \item \textbf{RQ2} ($N = 1{,}920$ leading-probe trials): MDE for $|\rho| = 0.064$. All observed correlations exceed this.
    \item \textbf{RQ3} ($N = 640$ per model): MDE for Cohen's $f = 0.056$ (small effects detectable).
\end{itemize}
